1. Rewrite the following rule using the font shortcut property:
blockquote {font-style: italic; font-size: 1.2em; line-height: 1.75em; font-family: times, serif;}

Answer: blockquote {font: italic 1.2em/1.75em times, serif}

2. What is the size of the em?

It is traditionally the horizontal length of a capital letter M in a given font size.

3. What does the browser do if you specify a font that is not stored on a user's computer?

It will look for the alternative fonts listed and if those are not stored either it will use the default.
